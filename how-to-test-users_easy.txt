./irssi/run_irssi_connect.sh [path-to-configfile] [nick_of_your_choice] 
    
    After running the script, it will run irssi and:
    will connect to localhost at 6667.
    will authenticate using the password - start - to connect to the server, so run the server with this password as well.
    User will be named as [nick_of_your_choice] for nick and "[nick_of_your_choice]-user" for user.

example for test usage, ran from ft_irc folder:

./irssi/run_irssi_connect.sh ./irssi/configs tester
